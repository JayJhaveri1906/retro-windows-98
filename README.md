## Key Features

* Boot Menu (with countdown).
* Blue Screen of Death (BSOD).
* Login And BSOD sound effect.
* Desktop (with Clock, Start menu press effect and Internet Explorer).
* Wokring Internet Explorer (Address Bar, Bookmarks, Home Button).
